STk File box help


This file box is an instance of the <File-box> composite class.

The Parents list contains current path exploded with one directory per line. Clicking on an item of this list allows a rapid access to a parent directory.

The Files list contains the files of the current directory. Special files (i.e dotted files) are shown iff the check button All files, situated on the bottom of the window, is set.

A file name can also be typed-in in the File name entry. Use the Tab key to perform automatic completion.
