S-scape: The STk Web Browser.

S-scape is a Web browser

  • Entirely written in Scheme;

  • Accepts a subset of the HTML 2.0 norm (no "forms")

  • Is used generally to access STk on line help:

    • (help)

    • (help "bell")

  • Allows the definition of Scheme "applets"

  • can browse remote pages (e.g. http://kaolin.unice.fr/STk/)
